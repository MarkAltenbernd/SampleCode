Word Indexer

Implemented in HTML

Requirements - HTML

  • A “word” is any sequence of capital letters A-Z and is not limited to just "dictionary words"
  • For any word with at least two different letters, there are other words composed of the same letters but in a different order
  • We can assign a number to every word, an index of where it falls in an alphabetically sorted list of all words made up of the same set of letters
  • Write a program that takes a word as input and computes its index in the related sorted list
  • A word may be up to 25 letters in length, with repeated letters permitted
  • The program should use no more than 1 GB of memory and take no more than 500 milliseconds to run
